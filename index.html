Privacy Policy for FireFighter-Alarm

Introduction

Thank you for using FireFighter-Alarm. Protecting your privacy is important to us. This Privacy Policy explains what information we collect and how we use it.

Data Collection and Usage

FireFighter-Alarm does not collect, store, or share any personal information beyond what is necessary for authentication.

If you choose to log in, authentication is handled securely via AWS Cognito. The following data may be used for login purposes:

  • Username or Email Address
  • Password

This information is only used to verify your identity. We do not store, share, or process this information beyond the login procedure.

Data Sharing

We do not share any user data with third parties.

Data Security

Authentication is handled using industry-standard security practices through AWS Cognito.

Changes to This Privacy Policy

If we make changes to this Privacy Policy, we will update this document accordingly.
